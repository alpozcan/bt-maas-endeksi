## Quality Metrics (each scored 1-10)
1. **Visual Hierarchy** — Clear focal point, natural eye flow, layered information
2. **Information Density** — Data-to-chrome ratio, chart readability, no wasted space
3. **Interactivity** — Hover/click/drag for detail, filter responsiveness, smooth transitions
4. **Animation Polish** — Scroll reveals well-timed, number easing, no janky transitions
5. **Typography & Spacing** — Consistent type scale, whitespace rhythm, aligned grid
